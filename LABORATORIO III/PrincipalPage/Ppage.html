Etiquetas semanticas estructurales mas importantes:

  • HEADER
  • NAV
  • MAIN
  • SECTION
  • FOOTER
  • ASIDE
  • ARTICLE

Algunas etiquetas semanticas y sus usos:

  • Article: Esta etiqueta se utiliza para definir una sección independiente y autocontenida de contenido dentro de una página web. Puede ser un artículo de una publicación, una entrada de blog, un comentario, una receta, entre otros.
  • Section: Se utiliza para dividir una página web en secciones lógicas y separadas. Por ejemplo, podemos utilizarla para separar los capítulos de un libro o los bloques temáticos de un artículo.
  • Aside: Esta etiqueta se utiliza para definir un contenido relacionado pero que no es parte del flujo principal de la página web, como un contenido secundario, un anuncio, una nota, una barra lateral, etc.
  • Figure: Se utiliza para representar un contenido independiente, como imágenes, gráficos, diagramas, vídeos, entre otros, que pueden ser acompañados de una leyenda o un texto alternativo.
